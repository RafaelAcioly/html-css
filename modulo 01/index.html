A tag "em" torna uma frase em itálico

A tag "mark" marca um texto

As tags "big" e "small" servem para aumentar o texto e torna-lo pequeno respctivamente, porém a "big" está entrando em desuso

Com a tag "del" Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado

Com a tag "ins" podemos marcar um texto como inserido para indicar que ele foi adicionado depois

A tag "sup" serve para inserir textos sobrescritos, coisas do tipo x20=3

A tag "sub" serve para inserir textos subexcritos, coisas do tipo H2O

A taga "code" deixa as letras todas com o mesmo espaçamento: PorExemplo PorExemplo

A tag "pre" coloca o texto (com indetação) no mesmo formato que está exposto no código Exemplo:

A tag "q", coloca aspas em um texto, dando siginifaco de citação. Exemplo: Como diria um pai de um amigo o computador é um burro rápido

A tag "blockquote" adiciona uma citação em uma quebra de linha e um espaçamento.

Citação em uma quebra de linha e um espaçamento. dentro do "blockquote" você pode adicionar um parâmetro citando o endereço de onde se pode achar a citação descrita, não serve como link mais vai servir aos mecanismos de busca do google.
